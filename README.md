文件上传服务 (File Upload Service)
基于 Gin 框架的文件上传服务，支持大文件上传、断点续传和文件管理功能。

功能特性
🚀 大文件上传 - 支持大文件分片上传

⏯️ 断点续传 - 上传中断后可恢复

📁 文件管理 - 查看、下载和删除已上传文件

🎨 响应式设计 - 适配各种屏幕尺寸

📊 实时进度 - 上传进度可视化展示

🛡️ 安全可靠 - 文件名校验和大小限制

技术栈
后端框架: Gin

前端模板: HTML5 + CSS3 + JavaScript

图标库: Font Awesome

HTTP客户端: axios

安装与运行
前提条件
Go 1.16+
